Monte Carlo Simulation

To the left is a program that I made that simulates the Monte Carlo algorithm which is a computer algorithm that determines the sampling rate that is needed for a batch of items to catch a certain percentage of bad batches. This is/can be used in an industrial setting such as assembly lines. This specific program has inputted a set percentage of things to be wrong and then makes random items in bad batches. Then it samples all the batches to see if it can find a bad batch.
